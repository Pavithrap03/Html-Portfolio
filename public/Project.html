Some of my Projects

Smart Car Parking using IoT - Internet of Things

Smart Car Parking System, leverages IoT technology to streamline parking management. Using sensors, the system detects available parking slots in real time and updates a cloud-based dashboard. This ensures efficient space utilization, reduces search time, and provides users with a seamless parking experience through automated monitoring and guidance.

Smart Street Light using IoT

My project, Smart Street Light System, utilizes IoT technology to optimize energy consumption. The system automatically adjusts street light intensity based on ambient light and motion detection. It integrates sensors and cloud connectivity to enable real-time monitoring, fault detection, and efficient energy management, contributing to smarter and more sustainable urban infrastructure.

Smart Vacuum Cleaner

My project, Smart Vacuum Cleaner, is an IoT-based automation system designed for efficient cleaning. It features RFID technology for room identification, cloud integration for tracking activities, and solar panels for self-charging. The system autonomously navigates, cleans, and optimizes energy usage, offering a sustainable and intelligent solution for modern housekeeping.
